查找包

Package Manager 窗口提供了多种方法来帮助您查找特定包：
(A) Packages 列表上下文 选择“上下文”确定哪些包出现在列表中。上下文可能是包的来源，例如注册表服务器、Asset Store 或 Unity 编辑器本身（对于内置包）；然而 In Project 上下文只显示那些已经安装在当前项目中的包，无论其来源。

例如，您可以选择 My Assets 上下文在列表中仅显示您可用的 Asset Store 资源包，或选择 In Project 上下文以仅显示项目中已安装的 Unity 包。
(B) Sort 根据名称、发布日期（仅限 Unity 包）、购买日期（仅限 Asset Store 资源包）或更新日期，以升序或降序对包的列表进行排序。

例如，如果您要查找具有最近更新但不记得其名称的 Unity 包，请尝试按更新日期降序排序并从上到下浏览列表即可找到。
(C) Asset Store Filters 选择如何使用以下这些组缩小 Asset Store 资源包列表的范围：

- Status__（隐藏，已弃用，无标签）
- Categories__ （2D、3D、音频、模板、工具等）
- __Labels__（您在 Asset Store 中定义的自定义标签）
(D) 搜索 使用搜索框按名称查找 Unity 包或 Asset Store 资源包。

这些控件有助于缩小列表视图中包的显示范围以及显示顺序。这样可以更轻松地找到您要查找的内容，或者在您不确切知道自己想要什么时帮助您浏览。

当您同时使用多个控件时，可以缩小列表中显示的匹配集。

找到您需要的包后，就可以从列表中找到特定版本。

查找特定版本

要查找某个特定的包版本，请执行以下操作：

1.在 Unity 中，打开 Package Manager 窗口（导航到 Unity 的主菜单，然后选择 Window > Package Manager）。

2.如果您正在寻找一个预览包，遵循以下步骤：

* 单击[高级设置](upm-ui.html#Advanced) ![](../uploads/Main/PackageManagerUI-settings.png) 图标，选择  __Advanced Project Settings__。

    ![Show preview packages](../uploads/Main/upm-docs.png)

    [Package Manager](class-PackageManager.html) 项目设置窗口出现。

* 在 **Advanced Settings** 组下面，启用 __Enable Preview Packages__ 选项。

* 关闭 **Project Settings** 窗口。任何[预览包](pack-preview.html)现在出现在列表中，带有 ![](../uploads/Main/iconPreview.png) 标识。

3.您可以浏览包列表或使用以下方法之一查找特定包：

* 按名称或关键字[搜索包](upm-ui-search.html)。
* [对列表进行排序](upm-ui-sort.html)（例如，按上次更新时间降序排序，以查找最近更新的包）。
* 对列表[应用过滤器](upm-ui-filter2.html)（仅限 Asset Store 资源包）。

4.在包列表中，单击包名称左侧的展开箭头图标。

![部分展开的列表仅显示当前安装的版本、推荐的更新以及 **See other versions** 链接](../uploads/Main/PackageManagerUI-AvailableUpdates.png)

如果有可用更新，则这些更新会显示在 **See all versions** 链接上方。

5.单击 See all versions 查看该包的所有可用版本列表。

此时将显示一个可滚动列表，其中包含所有可用版本。

![扩展列表显示所有可用版本](../uploads/Main/PackageManagerUI-AvailableVersions.png)

6.选择这些版本中的任何一个，可查看特定于该版本的详细信息。

![出现在详细信息窗格的版本，以及匹配列表中的选定版本的 **Update** 按钮](../uploads/Main/upm-docs3.png)

7.选择特定版本时，可以执行多种操作：

* 可以单击 **View documentation** 链接以打开与该版本匹配的[文档](upm-docs.html)。
* 可以单击 **View changelog** 链接以查看在创建包之后直至此版本期间进行的更改的列表。
* 可以单击 **View licenses** 链接以查看此包专门针对此版本的任何许可证信息。
* 可以查看此包是否[依赖](upm-dependencies.html)其他任何包，或者是否有任何其他包依赖此包。
* 如果此包已安装且包含示例资源，则可以单击 __Import__ 按钮以将其副本保存在项目的 `Assets` 文件夹下面。
* 如果还没有安装此包，则可以[安装](upm-ui-install.html)此版本。
* 如果已经安装了此包的另一个版本，则可以将此包[更新](upm-ui-update.html)到此版本。
* 如果以前已经安装了此包，但是不想再使用，则可以[删除](upm-ui-remove.html)此包。
