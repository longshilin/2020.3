搜索框

您可以使用 Package Manager 窗口顶部的搜索框查找与您输入的文本匹配的所有包。在您输入文本时，Package Manager 将更新包列表，显示当前上下文内匹配的每个包。例如，如果搜索框中输入 web，列表将显示不同的结果，具体取决于您选择的是 Built-in packages 列表上下文还是 Unity Registry 列表上下文：

可以按照包的名称或显示名称查找包。您可以搜索显示名称（也就是 Package Manager 窗口中显示的名称），对于 Unity 包，则可以搜索已注册包名称（以 com.unity 开头）。

搜索文本可以与包名称的开头、中间或结尾匹配。Package Manager 动态更新包列表。

注意：您可以搜索 Unity 包或 Asset Store 资源包。选择的列表上下文确定了 Package Manager 进行搜索的位置：

列表上下文： Package Manager 进行搜索的位置：
Unity Registry Unity 注册表中的包的列表
My Registries 为项目定义的任何作用域注册表
In Project 当前项目下的包缓存和 Packages 文件夹
My Assets 您的 Unity 帐户可用的 Asset Store 资源包的列表
Built-in packages Unity 注册表中的内置包的列表
