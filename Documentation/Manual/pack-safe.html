已验证包

已验证包是指通过了验证并可以在特定 Unity 版本上使用的包，也可以和其他已验证可用于该版本的包一起使用。经过验证可用于该 Unity 版本的包会在 Package Manager 窗口中的包旁边显示 指示符号。有关包的已验证状态和生命周期的更多信息，请参阅包状态和生命周期。
